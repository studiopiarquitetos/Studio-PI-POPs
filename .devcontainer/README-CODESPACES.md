# Como abrir no GitHub Codespaces

## Passo a passo (sem instalar nada no seu computador)

### 1. Abrir o Codespace
- Acesse o repositório no GitHub
- Clique no botão verde **`<> Code`**
- Clique na aba **`Codespaces`**
- Clique em **`Create codespace on claude/multi-agent-system-FVYg2`**

### 2. Aguardar a configuração
O Codespace vai:
- Instalar Python e Node.js automaticamente
- Instalar todas as dependências
- Iniciar o servidor na porta 7860
- **Abrir o link automaticamente no navegador** ✅

### 3. Autenticar o Claude CLI
Na primeira vez, o terminal vai pedir para fazer login:
```
claude login
```
Siga as instruções que aparecem (abre o navegador para autorizar).

### 4. Usar os agentes
Digite qualquer tarefa na interface e veja os agentes trabalhando!

---

> **Dica:** O Codespace fica ativo por até 30 dias.
> Você tem 120 horas/mês gratuitas no GitHub.
